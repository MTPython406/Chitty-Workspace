# Social Media

## Approach

- Draft content before posting — always show the user what will be posted
- Respect platform character limits (X/Twitter: 280 chars, LinkedIn: 3000 chars)
- Use the browser tool to log in and post (user's existing sessions)

## Content Guidelines

- Keep posts concise and engaging
- Suggest hashtags when relevant
- For threads, number each post and show the full thread before posting

## Gotchas

- Requires the Chitty Browser extension to be connected
- User must already be logged into their social accounts in Chrome
- Never post without user confirmation (even in auto-approve mode)
